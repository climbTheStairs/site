Welcome to Bruce's MBTI Site!

MBTI, also called the Myer-Briggs Type Indicator, is a personality-typing system I am interested in. Despite lacking in scientific evidence and sometimes being considered pseudoscience, it's can be fun to play around with, typing yourself, friends, or even fictional characters! The Myer-Briggs Type Indicator was first developed in the early twentieth century based on psychologist Carl Jung's theories by Katherine Briggs and her daughter Isabel Myers.

There are sixteen different personality types in the Myer-Briggs Type Indicator, with each of them being a four-letter combination:

  • ENTJ
  • INTJ
  • ESTJ
  • ISTJ
  • ENFJ
  • INFJ
  • ESFJ
  • ISFJ
  • ENTP
  • INTP
  • ESTP
  • ISTP
  • ENFP
  • INFJ
  • ESFP
  • ISFP

There are two ways to type using the Myer-Briggs Type Indicator: using letter dichotomies and using cognitive functions. Click the links to find out about each of these methods!
